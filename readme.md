## 项目立项

本项目基于Ruoyi-vue-plus v5.x版本修改拓展 [RuoYi-Vue-Plus](https://gitee.com/dromara/RuoYi-Vue-Plus)

这是一个基于**Spring Boot 3的后端通用接口案例项目**，汇总了实际开发中常见的需求代码**片段**。**该项目旨在帮助开发者快速搭建后端服务，解决实际业务需求中的常见问题。**

此项目不是一个脚手架，只是一些通用解决方案的一个**片段汇总**，不适合从0开发去用这个系统，而是你要新开发功能时，没有思路可以参考这里面的其中某个模块，或者某个文件。

该项目会保留U-Easyproject的所有代码，并且在docs文件夹里讲解所有的代码。

其他的新加代码，以及数据库等，我会和原项目分开。

以ruoyi-xx的模块为**原项目代码**

以u-xx的模块为**新代码**
