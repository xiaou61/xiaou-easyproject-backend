## About This Project

**The English documentation may not be updated in real-time.**

This is a backend universal interface case project based on Spring Boot 3, summarizing common code snippets for actual development needs. The project aims to help developers quickly set up backend services and address common issues in actual business requirements.

I started this project on a whim, but I plan to stick with it for a long time.

As a third-year college student who has worked on many projects, I've noticed that a significant portion of the code in these projects is highly repetitive. Additionally, code snippets found online are either too simplistic and not suitable for actual development, or they are paid resources, etc.

So, I thought, why not compile all the good examples and demonstrations I've encountered?

With this idea in mind, this project was officially established.

Of course, my personal coding skills are limited, and I try my best to refine each example I publish.

Many of the codes in this project are excellent examples from some open-source or tutorial projects I've seen. I will mark the sources in the acknowledgment list later.

## Tech Stack

**[Spring Boot 3](https://spring.io/projects/spring-boot)**: Core framework for building backend services

**[Lombok](https://projectlombok.org/)**: Plugin to simplify Java code

**[Fastjson](https://github.com/alibaba/fastjson)**: High-performance JSON library

**[Knife4j](https://github.com/xiaoymin/swagger-bootstrap-ui)**: Enhanced Swagger interface documentation tool

**[Guava](https://github.com/google/guava)**: Google's core Java library

[**Javax Servlet**](https://mvnrepository.com/artifact/javax.servlet/javax.servlet-api): Servlet API

**[Hutool](https://hutool.cn/)**: Java utility library

**[Kaptcha](https://github.com/penggle/kaptcha)**: CAPTCHA generation library

[**Aliyun OSS**](https://github.com/aliyun/aliyun-oss-java-sdk): Alibaba Cloud Object Storage Service

[**X-File-Storage**](https://github.com/dromara/x-file-storage): File storage framework

**[Spring Data Redis](https://spring.io/projects/spring-data-redis)**: Redis data access framework

**[JJWT](https://github.com/jwtk/jjwt)**: JSON Web Token library

[**MySQL Connector**](https://mvnrepository.com/artifact/mysql/mysql-connector-java): Driver for connecting to MySQL databases

**[Druid](https://github.com/alibaba/druid)**: Alibaba's database connection pool

**[MyBatis-Plus](https://github.com/baomidou/mybatis-plus)**: An enhanced MyBatis framework

**[MyBatis Spring](https://github.com/mybatis/spring-boot-starter)**: Library for integrating MyBatis and Spring

**[Spring Data Redis](https://github.com/spring-projects/spring-data-redis)**: Spring framework data access project for Redis

**[Apache Commons Pool](https://commons.apache.org/proper/commons-pool/)**: Apache's general object pool library
